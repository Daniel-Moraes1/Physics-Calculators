One Dimension Elastic Collisions

In a Perfectly Elastic Collision, energy in a system must be conserved before and after a collision. Assuming the masses of the objects do not change, the only way for this to be possible is if the velocity at which the objects approach is equal to the velocity at which they separate. We can use this to calculate the velocity of an object, given the approach velocities of the objects and the final velocity of one object.

Input 3 values of data into the table, and the last one will fill when you hit solve. The right box will display the change in energy before and after the collision. If you only know the approach velocity and not the velocity of each object, input 0 for the initial velocity of object 1 and the approach velocity as the velocity of object 2.

Input Collision Data

Velocity of Object 1
Velocity of Object 2
Final Velocity Object 1
